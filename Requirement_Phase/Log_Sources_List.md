# Log Sources List

## 1. Firewall Logs
Used to monitor inbound and outbound network traffic.

## 2. Windows Event Logs
Used to track login activity, system events, and security events.

## 3. VPN Logs
Used to monitor remote user login activity and access attempts.

## 4. IAM Logs
Used to track authentication and user account activities.

## 5. System/Application Logs
Used to monitor application-level events and errors.
