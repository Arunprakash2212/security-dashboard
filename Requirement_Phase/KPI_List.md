# KPI List

## 1. Alert Count
Total number of security alerts generated.

## 2. Severity Distribution
Number of alerts categorized as Low, Medium, High, and Critical.

## 3. Top Source IPs
IP addresses generating the highest number of alerts.

## 4. Log Volume Trend
Number of logs received over a selected time period.

## 5. Incident Response Time
Time taken to identify and respond to alerts.
