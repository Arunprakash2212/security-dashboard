# Project Scope

## Project Title
Security Dashboard Creation

## Objective
To develop a basic SOC dashboard that centralizes security alerts and provides real-time visibility of security events.

## Scope
- Collect logs from Firewall, Windows, and VPN.
- Create basic detection rules (e.g., brute force detection).
- Classify alerts by severity.
- Visualize alerts using dashboard panels.

## Out of Scope
- AI/ML-based detection
- Automated incident response
- Full enterprise deployment

## Expected Outcome
A functional SOC dashboard with basic alert monitoring and reporting capability.
