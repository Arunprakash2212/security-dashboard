# Functional Requirements

## 1. Log Collection
- Collect logs from Firewall, Windows Event Logs, and VPN.
- Ensure logs are properly parsed and searchable.

## 2. Alert Creation
- Create basic detection rules (e.g., brute-force login detection).
- Assign severity levels (Low, Medium, High, Critical).

## 3. Dashboard Visualization
- Display alert trends over time.
- Show severity distribution.
- Display top source IP addresses.
- Show log source breakdown.

## 4. Filtering & Reporting
- Allow filtering by time range and severity.
- Support basic report generation.
