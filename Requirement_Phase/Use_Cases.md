# Use Cases

## 1. Brute-Force Login Detection
Detect multiple failed login attempts from the same IP within a short time period.

## 2. Suspicious Login Detection
Identify successful logins after multiple failed attempts.

## 3. Abnormal Outbound Traffic Detection
Detect unusually high outbound network traffic from a host.

## 4. High-Severity Alert Monitoring
Display and prioritize critical security alerts on the dashboard.
