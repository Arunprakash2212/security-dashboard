# SIEM Installation Document

## 1. SIEM Tool Used
Splunk Enterprise (Local Lab Setup)

## 2. System Requirements
- OS: Windows 10 / Linux
- RAM: Minimum 4 GB (8 GB recommended)
- Storage: Minimum 20 GB free space
- Internet connection for download

## 3. Installation Steps

1. Download Splunk Enterprise from the official website.
2. Run the installer and accept the license agreement.
3. Choose installation directory.
4. Set administrator username and password.
5. Complete installation and launch Splunk Web.
6. Access SIEM using: http://localhost:8000

## 4. Initial Configuration

- Logged in as admin user.
- Verified Splunk service is running.
- Checked default index creation.
- Enabled receiving data (if needed).

## 5. Installation Verification

- Confirmed Splunk Web interface is accessible.
- Verified system status is active.
